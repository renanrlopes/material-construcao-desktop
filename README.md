# 🏗️ Asi & Obra - Sistema de Gestão de Materiais

Este é um sistema robusto de gestão de inventário desenvolvido para uma empresa de materiais de construção. O projeto engloba o controle de produtos, gestão de estoque por lotes, auditoria de movimentações (baixas) e controle de acesso baseado em cargos.

## 🚀 Tecnologias Utilizadas

* **Frontend**: [Angular CLI] - 18.2.14.
* **Backend & Database**: [Firebase](https://firebase.google.com/).
* **Autenticação**: Firebase Authentication com suporte a múltiplos perfis (Admin, Estoquista, Usuário).
